XAMPP Setup Steps (for Windows)
1. Start Services

In your XAMPP Control Panel:

Click Start next to Apache

Click Start next to MySQL
You should see both turning green. 





2.  Open phpMyAdmin

Open your browser

Go to: http://localhost/phpmyadmin




Step-by-Step: Create and Select the Correct Database
1.	Go to the phpMyAdmin home page

In the left panel, click on "New" just under the phpMyAdmin logo.

2. Create the new database

In the "Create database" field, type exactly:

roster

Leave collation as default (or choose utf8_general_ci).

Click Create.

You should now see the new database named roster appear on the left panel.




Step 2: Select the roster Database

Click on roster in the left sidebar.

Then click the SQL tab at the top.





Step 3: Paste in the SQL Code

Once you're in the SQL tab of the roster database, paste in this code (just the first block to test):

DROP TABLE IF EXISTS Member;
DROP TABLE IF EXISTS `User`;
DROP TABLE IF EXISTS Course;

CREATE TABLE `User` (
    user_id     INTEGER NOT NULL AUTO_INCREMENT,
    name        VARCHAR(128) UNIQUE,
    PRIMARY KEY(user_id)
) ENGINE=InnoDB CHARACTER SET=utf8;

CREATE TABLE Course (
    course_id     INTEGER NOT NULL AUTO_INCREMENT,
    title         VARCHAR(128) UNIQUE,
    PRIMARY KEY(course_id)
) ENGINE=InnoDB CHARACTER SET=utf8;

CREATE TABLE Member (
    user_id       INTEGER,
    course_id     INTEGER,
    role          INTEGER,
    CONSTRAINT FOREIGN KEY (user_id) REFERENCES `User` (user_id)
      ON DELETE CASCADE ON UPDATE CASCADE,
    CONSTRAINT FOREIGN KEY (course_id) REFERENCES Course (course_id)
      ON DELETE CASCADE ON UPDATE CASCADE,
    PRIMARY KEY (user_id, course_id)
) ENGINE=InnoDB CHARACTER SET=utf8;

Then click Go.

You should see ‚ÄúQuery OK‚Äù and all 3 tables created.
