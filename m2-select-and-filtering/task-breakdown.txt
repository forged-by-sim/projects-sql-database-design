Step 1: Create the Ages table

Run this SQL command:

CREATE TABLE Ages ( 
  `name` VARCHAR(128), 
  age INTEGER
);



Step 2: Clear the table and insert the data

Run these commands one by one:

DELETE FROM Ages;

INSERT INTO Ages (`name`, age) VALUES ('Akam', 13);
INSERT INTO Ages (`name`, age) VALUES ('Oisin', 21);
INSERT INTO Ages (`name`, age) VALUES ('Mhirren', 34);
INSERT INTO Ages (`name`, age) VALUES ('Aleeshmah', 25);
INSERT INTO Ages (`name`, age) VALUES ('Nana', 33);
INSERT INTO Ages (`name`, age) VALUES ('Blythe', 28);


Step 3: Run the hash function and sort

SELECT sha1(CONCAT(`name`, age)) AS X FROM Ages ORDER BY X;

This will return a list of SHA1 hashes.



Step 4: Get the first row's SHA1

Look at the first row of the result. The value in column X is the answer you need.
